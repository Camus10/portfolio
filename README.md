## ✨ Features

- 🎯 **Interactive Particle Background** - Dynamic particle network animation in the hero section
- 🖱️ **Custom Cursor** - Custom cursor with smooth hover effects and trail animation
- 🤖 **AI-Powered Chatbot Assistant** - Interactive chatbot with pre-defined responses based on CV/resume
- 🎨 **Smooth Animations** - Scroll-triggered reveal animations and smooth transitions
- 📱 **Fully Responsive** - Optimized for desktop, tablet, and mobile devices
- 🖼️ **Image Modal Viewer** - Click project screenshots to view in fullscreen lightbox
- ⚡ **Back to Top Button** - Smooth scroll navigation with floating action button
- 📊 **Animated Statistics** - Counter animation for experience, technologies, and companies
- 🎮 **Gaming-Inspired Design** - Terminal-style preloader and system prompt aesthetic
